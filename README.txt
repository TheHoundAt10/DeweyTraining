# Dewey Decimal Sorting Mini-Game

## Overview

The Dewey Decimal Sorting Mini-Game (Replace Books) is a simple educational game where users can practice sorting books according to the Dewey Decimal Classification system. The game provides a user interface with rows of books, each with a book image, a call number, and up/down buttons for reordering.

## Features

- Random generation of Dewey Decimal numbers and book images.
- Scoring system based on correct Dewey Decimal sorting.
- Sound effects for user interactions.
- User-friendly interface with easy-to-understand controls.

## Installation

1. **Clone the Repository**: Clone this repository to your local machine using Git:



2. **Open in Visual Studio**: Open the project in Visual Studio (or your preferred C# development environment).

3. **Build and Run**: Build and run the project to play the mini-game.

## Usage

- Launch the application.
- Reorder the books by clicking the up/down buttons.
- Try to sort the books in the correct Dewey Decimal order.
- Click the "Check" button to evaluate your score.
- View your total moves and final score.
