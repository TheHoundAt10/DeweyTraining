# Dewey Decimal Training App

## Overview

The Dewey Decimal Training App provides two games. "Replace Books" is a simple educational game where users can practice sorting books according to the Dewey Decimal Classification system. The game provides a user interface with rows of books, each with a book image, a call number, and up/down buttons for reordering.

"Identify areas" - Users can become more familiar with the Dewey Decimal System's top-level categories by playing this interactive learning game. Users can participate in the entertaining match-the-columns game, where they have to correctly match up the call numbers with the corresponding descriptions. The game has randomly generated questions, gamification components that encourage practise, and a user-friendly UI for a fun learning experience.

## Features

@ Replace Books:
- Random generation of Dewey Decimal numbers and book images.
- Scoring system based on correct Dewey Decimal sorting.
- Sound effects for user interactions.
- User-friendly interface with easy-to-understand controls.

@ Identify Areas:
- Random generation of Dewey Decimal call numbers and their respective descriptions.
- Scoring system based on correct match-ups.
- Sound effects for user interactions.
- User-friendly interface with easy-to-understand controls.
